# Demo V11 變更摘要

## 本版主軸：患者管理實作化

V11 開始把「患者管理」從入口按鈕改成可操作流程。

## 新增功能

1. 新增個案表單。
2. 初診建檔表單。
3. 複診登記紀錄。
4. 個案資訊查詢。
5. 列管事項紀錄。
6. 客服關懷紀錄。
7. 患者事件紀錄表 patient_events。
8. 患者檔案內可新增初診、複診、客服、居家、列管與備註紀錄。
9. 新增列管、客服、居家事件時，會同步建立待辦任務。
10. 患者管理仍使用示範資料，不可輸入真實病患資料。

## 必做 SQL

請執行：

supabase/v11_patch.sql

執行後會新增：

patient_events

## 測試建議

1. 進入「患者管理」。
2. 建立一筆新增個案。
3. 切換到初診建檔，再建立一筆示範患者。
4. 選擇既有患者，新增複診登記紀錄。
5. 新增客服關懷或列管事項，確認任務中心有新增待辦。
6. 點「查看檔案」，確認患者事件紀錄有出現。
