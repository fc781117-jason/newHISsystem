
# NEW HIS Demo V15 修正說明

## V15 修正重點

1. 登入後先選擇「今日上班院區」，再進入系統。
2. 搜尋與篩選保留日期與搜尋，不再把院區放在一般篩選裡。
3. 今日流程看板按鈕由「完成本站 → 下一步」改為「完成」。
4. 已完診者不顯示完成按鈕。
5. 完成流程時會新增 patient_events 流程紀錄。
6. 任務中心新增篩選：優先、類型、組別、狀態、到期日。
7. 工作指揮中心新增「休假／代理／審核提醒」。
8. 請假與代理改成正式表單：申請類型、假別、起迄日期時間、指定代理人、原因。
9. 請假流程：申請人送出 → 代理人同意 → 主管核准 → 寫入班表。
10. 支援取消申請。
11. 月排班改成組別月班表格，依時段顯示姓氏。
12. 支援個人班表查詢與列印／另存 PDF。
13. 工時統計改成折疊。
14. 強化必要資訊揭露：一般人看自己的任務與代理，主管看整體待審。

## 必做 SQL

請執行：

supabase/v15_patch.sql

## 測試建議

1. 登入後先選今日院區。
2. 進入今日流程，確認按鈕為「完成」。
3. 進入任務中心，測試篩選。
4. 進入請假與代理，建立請假／卡位／調班申請。
5. 用代理人帳號測試同意代理。
6. 用主管帳號測試核准後是否寫入班表。
7. 進入月排班與調班，查看組別月班表與個人班表。
