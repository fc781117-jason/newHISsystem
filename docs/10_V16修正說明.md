# NEW HIS Demo V16 修正說明

## 本版修正重點

1. 工作指揮中心儀表板改為可點擊連結：
   - 今日患者 → 患者管理
   - 預約掛號 → 掛號與初複診
   - 照護工單 → 居家／製衣／客服
   - 高優先 → 任務中心
   - 休假／代理待處理 → 請假與代理
   - 未簽收公告 → 公告與簽收

2. 今日流程看板：
   - 「完成」保留。
   - 新增「完診」按鈕，讓患者可不經後續拿藥／處方流程直接結束。
   - 完成與完診皆會寫入患者事件紀錄。

3. 任務中心：
   - 保留 V15 的優先、類型、組別、狀態、到期日篩選。

4. 請假與代理：
   - 新增常見原因下拉。
   - 主管可直接看到核准按鈕。
   - 保留代理人同意與取消申請。

5. 月排班與調班：
   - 移除「休／假／代理」列，只顯示當日上班者。
   - 個人班表改成與組別班表相同的表格式，不再使用條列式。
   - 依組別或個人切換後，輸出 PDF 只輸出目前班表區塊。
   - 班表外框與內部分隔線加粗。
   - 點擊人員姓氏可開啟彈窗設定當日任務組別與任務說明。
