# Demo V8 變更摘要

## 這版修正重點

1. 首頁移除「原系統功能整併摘要」與「舊系統功能整併方向」。
2. 原系統功能整併內容只保留在「行政營運 → 功能整併總覽」。
3. 手機版上方「院區、日期、搜尋」改成可折疊的「搜尋與篩選」。
4. 院區、日期、搜尋欄位統一高度與寬度。
5. 首頁移除重複的登入者身份卡，保留右上角使用者資訊。
6. 首頁儀表板改成短卡片，手機版兩欄排列，避免頁面過長。
7. 今日必辦改成可依「高中低優先、狀態、組別」排序。
8. 今日必辦內容改成折疊式清單。
9. 主管與最高管理員可看到各組未完成事項統計。
10. 一般角色只看自己組別或指派給自己的任務。
11. 人員與權限管理新增「組別」與「預設院區」可調整。
12. 保留月排班與調班功能。

## 不需要新增 SQL

V8 使用既有 `profiles.group_name` 與 `profiles.default_clinic` 欄位，不需要新增資料表。
