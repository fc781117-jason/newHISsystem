# NEW HIS Demo V17 修正說明

## 本版重點

1. 「請假與代理」與「月排班與調班」整併為「班表管理系統」。
2. 班表管理系統內分成：
   - 月班表
   - 請假／代理／卡位
   - 調班申請
3. 月班表支援手動排班：
   - 點空白格新增人員
   - 點人員姓氏可修改、移動、刪除
   - 可指定院區
   - 可指定任務組別
   - 可輸入備註
4. 自動產生整月排班後，人員仍可手動調整。
5. 調班申請改成表單，不再是隨機產生。
6. 主管可核准，也可在緊急情況下做「主管強制指定」。
7. 人員與權限管理新增「額外功能權限」：
   - 建議採角色權限 + 個人例外授權
   - 不建議在每個功能頁分散設定權限
8. PDF 列印維持 A4 橫式，並只列印目前班表區塊。
