# NEW HIS系統-診所營運流程整合系統 Demo V17

V17 是針對班表管理、手動排班、調班申請與個人額外權限的修正版。

## 必做

請到 Supabase 執行：

```text
supabase/v17_patch.sql
```

## 更新方式

1. 上傳覆蓋 GitHub 專案
2. Commit
3. Vercel 自動部署
4. Supabase 執行 v17_patch.sql
5. 到 Vercel 更新 VITE_APP_TITLE：

```text
NEW HIS系統-診所營運流程整合系統 Demo V17
```
